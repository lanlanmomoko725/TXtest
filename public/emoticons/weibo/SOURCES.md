# Weibo emoticon sources

- Base emoticon mapping and image assets: `weibo-emoticon@1.1.5`, ISC license.
- `[666]` compatibility image: Sina static expression asset, downloaded from
  `https://face.t.sinajs.cn/t4/appstyle/expression/ext/normal/6c/2022_666_thumb.png`.

The application serves copied assets locally from `/emoticons/weibo/` and does
not load these images from a third-party CDN at runtime.
